# Tricolor Theme

The Explorer uses a quiet tricolor palette inspired by Fluminense: burgundy, green, and white. The theme is intentionally subtle and project-appropriate: it functions as an identity accent, not as sports branding.

Palette used:

- Burgundy: `#870A28`
- Green: `#00613C`
- White: `#FFFFFF`
- Soft background: `#F8F5EF`
- Ink: `#171313`
