# Welcome to the Terminal Universe

Not all terminals are the same.

Some are minimal. Some are extensible. Some are fast. Some are historical. Each one represents a different way of thinking about interaction with a system.

## Explore Like a Periodic Table

Just like elements in chemistry, terminal emulators can be organized by their properties: how they render, how they evolve, how they connect to systems, and how they are designed.

## Two Ways to Explore

**Explorer Mode:** intuitive, visual, beginner-friendly.

**Scientific Mode:** full ontology, precise classifications, reproducible structure.
