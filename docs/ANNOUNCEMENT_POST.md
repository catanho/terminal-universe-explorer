# Announcement Draft

Title: Terminal Universe Explorer — a periodic table of terminal emulators

I’ve been working on a project to map the design space of Unix/Linux terminal emulators.

Instead of just listing terminals, this tries to answer:

- how are terminals different, structurally?
- what design philosophies do they follow?
- how have they evolved over time?

The result is a curated dataset, a multidimensional ontology, and an interactive periodic-table-style explorer.

It is hybrid by design: a formal classification system underneath, with a visual and intuitive interface on top.
