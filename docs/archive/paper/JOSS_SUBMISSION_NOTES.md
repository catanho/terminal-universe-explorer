# JOSS Submission Notes

## Candidate title

Terminal Universe Explorer: A Periodic Table of Terminal Emulator Design

## Summary

Terminal Universe Explorer is a static, open-source web application and dataset for exploring a hybrid ontology of terminal emulator design. It provides a curated dataset of 79 terminal emulators, formal validation scripts, and a public interface with Explorer and Scientific modes.

## Statement of need

Terminal emulators are central to Unix/Linux workflows, yet available comparisons are often informal, fragmented, or narrowly technical. Terminal Universe Explorer provides a reproducible data model and accessible visualization for understanding terminal emulator diversity, lineage, and design trade-offs.
