# Release Notes — v2.3 Public Alpha

This release prepares Terminal Universe Explorer for public sharing.

## Added

- Enhanced static Explorer prototype with:
  - Explorer cards
  - Scientific Mode toggle
  - periodic-group overview
  - category, status, ecosystem, and search filters
- `package.json` with simple `start` and `validate` commands.
- MIT license text.
- Data/text license recommendation note.
- Public-facing repository map and quick-start instructions.

## Preserved

- The v2.2 release-candidate data model.
- The 79-record pruned terminal-emulator scope.
- Formal/public hybrid split.

## Known limitations

- The Explorer remains a static prototype, not a full production app.
- Source verification depth can be improved before journal submission.
- DOI metadata should be finalized only after author/order/license decisions are complete.
