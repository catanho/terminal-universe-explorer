# Terminal Universe Explorer v2.4 — Tricolor Public Launch Layer

This release adds the public launch layer and a subtle tricolor visual identity.

## Added

- Tricolor Explorer theme inspired by Fluminense: burgundy, green, and white.
- Improved app hero copy and onboarding language.
- Explorer Mode and Scientific Mode explanatory cards.
- Reset Filters button.
- Improved empty state copy.
- Manifesto, landing page copy, announcement post, UI copy layer, and theme documentation.

## Design principle

The theme is intentionally subtle: it gives the project an identity without turning the Explorer into sports branding. The ontology remains the center of the interface.

## Dataset

No dataset changes from v2.3. The dataset remains 79 records with no known scope violations.
