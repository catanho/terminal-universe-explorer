# Theme System v2.5

The Explorer now supports two visual modes:

- **Light Mode**: documentation-like, bright, readable, public-friendly.
- **Dark Mode**: terminal-inspired, focused, lower-glare, optimized for long exploration.

Both modes preserve the tricolor identity palette:

- Burgundy: structure, headings, emphasis.
- Green: interaction, active state, scientific/ontology accents.
- White/off-white: clarity and contrast.

The implementation uses CSS custom properties, `data-theme`, `prefers-color-scheme`, and `localStorage`.
