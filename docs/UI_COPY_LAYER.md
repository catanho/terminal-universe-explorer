# Terminal Universe Explorer — UI Copy Layer

This copy layer makes the hybrid ontology visible in the interface: formal enough for reproducibility, approachable enough for public exploration.

## App Identity

**Title:** Terminal Universe Explorer  
**Subtitle:** A periodic table of terminal emulator design

## Modes

### Explorer Mode
Visual, simplified view for discovering terminals and their relationships.

### Scientific Mode
Full ontology view with detailed attributes, lineage, confidence, and source-backed data.
